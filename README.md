# Sales ETL Pipeline (Python)

## Overview
This project demonstrates an end-to-end ETL (Extract, Transform, Load) pipeline using Python and Pandas.

## ETL Workflow
- Extracted raw sales data from CSV files
- Cleaned and validated records
- Removed failed transactions
- Handled missing values
- Generated payment-wise revenue summary

## Tech Stack
- Python
- Pandas
- Google Colab

## Output
- Processed sales dataset
- Payment-wise revenue summary

## Use Case
Simulates real-world Data Operations and ETL workflows.
